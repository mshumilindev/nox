# NoxMac

Canonical macOS app target for Nox. Sources live in `Nox/` (wired from `Nox.xcodeproj` as `Apps/NoxMac/Nox`).

- App lifecycle, adapters, SQLite stores, mesh transport, notifications
- SwiftUI / AppKit features

Shared domain types and contracts ship from `Packages/`. See `Docs/PLATFORM_MIGRATION.md`.
